CertShield – Digital Certificate Verification Mediator

📌 Project Description

CertShield is a digital certificate verification system built for Governance & E-Administration.

It allows government departments, universities, banks, and employers to verify certificates instantly and securely.

The system prevents fake certificates and reduces manual verification delays.



🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- File Handling: Multer
- Security: SHA-256 Hashing
- Database: JSON-based storage (certificates.json),mysql



🚀 Features

🔐 1. Government Admin Login
Secure login portal for authorized government officials to register and manage certificates.

🏢 2. Verifier Login
Dedicated access for universities, banks, employers, and other departments to verify certificates.

📄 3. Certificate Registration & Upload
Admins can upload certificate details and supporting documents (PDF, JPG, PNG).

🔎 4. QR Code Generation
Each certificate is assigned a unique QR code linked to its verification ID.

🔒 5. SHA-256 Secure Hashing
Tamper-proof hash generation ensures certificate integrity and prevents modification.

⚡ 6. Instant Certificate Verification
Verifiers can instantly validate certificates using QR scan or Certificate ID.

🛡️ 7. Fraud Detection
System automatically detects and flags altered or invalid certificates.

📁 8. Structured Digital Record Storage
Certificates are securely stored in structured JSON-based records for quick access.
